One of this values:

- **ul**: top-left
- **uc**: up-center
- **ur**: up-right
- **cl**: middle-left
- **cc**: center
- **cr**: middle-right
- **ll**: bottom-left
- **lc**: bottom-center
- **lr**: bottom-right

This values is not used since 2.3:

- ul_rsl
- uc_rsl
- ur_rsl
- cl_rsl
- cc_rsl
- cr_rsl
- ll_rsl
- lc_rsl
- lr_rsl
